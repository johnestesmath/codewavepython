    Section 2.3 User Input

    A big part of programming involves asking the program’s user for some sort of input.
    1. username and password
    2. setting preferences
    3. their favorite color
    We can easily interact with a program user with input(). Use the input() function to ask the user a question and get their answer.

    Remark 2.3.1.

    The input() function always gives us (or returns) a string. If you want a number, convert it using int() or float().
    Use input() is an exteremely valuable tool to add to your growing, programming toolkit!

    Subsection 2.3.1 Try It Yourself: Strings and Inputs

    Try the following exercises. Don’t be afraid of making errors. That’s how we learn.

    Checkpoint 2.3.2. Making Introductions.

    Ask a user for (1) their name, (2) their favorite animal, and (3) a place they would like to visit. Then summarize their answers with a sentence.

    Checkpoint 2.3.3.

    1. convert the sentence to lowercase
    2. count the number of times the letter ’s’ appears
    3. replace all spaces with an underscore ’_’
    4. print the resulting sentence.
